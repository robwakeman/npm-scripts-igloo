# npm-scripts-igloo

A build process using npm scripts with Sass and Babel.

Igloo is just a silly name, because I've thrown in an igloo icon from Font Awesome.

[Demo](http://dev.robwakeman.com/npm-scripts-igloo/)

## How it works

First build CSS:

- compile
- prefix
- compress

```javascript
npm run build-css
```

Then to start dev:

```javascript
npm start
```
